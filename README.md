# Hababi Lively Wallpaper
A custom made wallpaper I made for my new laptop. Compatible with [Lively Wallpaper](https://github.com/rocksdanister/lively). Licensed with GNU GPLv3, uses jQuery and some third-party images (read Attributions.txt for more info).

## Use
Put contents of this repo into a zip and then drag and drop into your Lively Wallpaper app.
